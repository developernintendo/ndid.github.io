      External Links

        • NDID

          System for management of Nintendo Developer IDs (NDIDs), the credentials used to access systems including the Developer Portal, OMAS, NMAS, IMAS, BMAS, and SMAS.

        • OMAS

          The Online Title Management System (OMAS) is where you apply for the use of network services. Once a title's application form has been accepted, Nintendo's servers are configured and the requester is notified when each step is complete. Please use OMAS2 for Nintendo Switch applications and the legacy OMAS website for Nintendo 3DS series and Wii U applications.

        • SLIM

          The Software Legal Information System (SLIM) is a system for developers of Nintendo Switch applications to create "software legal information" like who users should contact with inquiries and IP markings.

        • NMAS / NMAS2

          The NEX Server Management System (NMAS) allows management of NEX servers, Nintendo-provided servers which provide matchmaking, ranking, and data storage services to titles.

        • IMAS

          The Item Management System (IMAS) manages licensing information for add-on content (downloadable content) and services used by the Nintendo 3DS series and Wii U systems.

        • NCMS

          The Nintendo Content Management System (NCMS) is a system for submitting various types of information required during Nintendo eShop distribution. From here, you can adjust prices as well as the distribution date of your title and videos on Nintendo eShop.

        • Save Data Report

          The Save Data Report web tool provides a framework for obtaining users' save data and writing back corrected save data to the user's Switch console.

        • BMAS

          The Ban Management System (BMAS) manages corrective actions toward users and devices, including online service bans.

        • SMAS

          The Service stats Management System (SMAS) allows access to statistical data regarding online functionality including NEX.

        • SST

          Shop Sales Tool provides Nintendo eShop sales and download information for digital titles after the title is released. Access the tool for purchases and downloads by item, price, country and time period.

        • BCAT

          The BCAT Management Tool is a system that distributes game news and data to the Nintendo Switch. There are multiple tools for each server environment.

        • Play Reports

          The Play Report web tool provides a framework for obtaining and using report data that applications send to server infrastructure.
